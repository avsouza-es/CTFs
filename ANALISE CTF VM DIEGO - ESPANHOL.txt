			Versión de algunos servicios: SMB-2
		- Posibilidades de prueba:
		- 3389: - Metasploit: Existe un módulo llamado auxiliary/scanner/rdp/rdp_login que escanea credenciales vía diccionario o brute password contra el puerto 		RDP. Puede descubrir credenciales débiles
		- Fuerza bruta vía Hydra: Una herramienta como Hydra lanzada sobre el puerto 3389 puede probar varias combinaciones hasta descubrir la contraseña si es 		débil.
		- Exploit CVE-2019-0708: Metasploit también tiene un exploit para esta vulnerabilidad que permite ejecutar código si se aplica a esa versión de RDP. Esto 		daría al sistema una shell remota.
		- Verificar el fallo En concreto, el parche que soluciona CVE-2019-0708 en Windows Server 2012 R2 es KB4534310, que se incluye en las actualizaciones 			acumulativas de noviembre de 2019. Sin embargo, no parece posible explotar este fallo, ya que la versión de WS es
